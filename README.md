# Wild Explorer

双语动物科普小站。数据存放在 `data/animals/`（一物一文件）。

---

## 常用命令

| 操作 | 命令 |
|------|------|
| **添加动物** | 将 AI 生成的 JSON 数组粘贴到 `_draft_animals.json`，然后执行：<br>`node import_animals.js` |
| **添加照片** | 为尚未有图片的动物下载照片：<br>`node download_images.js` |
| **更新动物列表** | 按分类刷新根目录的 `ANIMALS_LIST.md`：<br>`node update_animals_list.js` |

详细流程见 [docs/ADD_ANIMAL_WORKFLOW.md](docs/ADD_ANIMAL_WORKFLOW.md)。
